# Figma to Static Website

## Overview
This project is a static website created by converting a Figma design into
a fully functional web page using core front-end technologies.
The goal of this project was to understand real-world UI implementation
and improve layout, styling, and responsiveness skills.

## Features
- Pixel-accurate UI based on Figma design
- Clean and responsive layout
- Interactive elements using JavaScript
- Cross-browser compatible static website

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Figma (UI reference)

## Development Approach
- Analyzed Figma design structure and components
- Converted design into semantic HTML
- Used CSS for layout, spacing, and responsiveness
- Added JavaScript for basic interactivity

## Learning Outcomes
- Understanding of design-to-code workflow
- Improved CSS layout and positioning skills
- Practical experience in translating UI designs into code
- Better code organization and readability

## How to Run
1. Clone the repository
2. Open 'index.html' in any modern browser
3. View and interact with the static website
